

### 第一梯队：区分“纯小白”与“入门学习者”（HSK 1-2 级）
*这一阶段的核心是考察用户是否摆脱了“英语单词直译成中文”的本能。*

**1. [G_Adjective_Predicate] 形容词谓语句（“是”的误用）**
*   **区分逻辑**：英语中说 "I am busy"，老外本能会说“我是忙”(❌)。只有掌握了“我很忙 / 他不累”(✅)，才算真正跨入中文门槛。
*   **测试场景**：让 Agent 问“你今天觉得累吗？”看用户回答是否带“是”。

**2.[G_WordOrder_TimeLoc] 时间/地点状语前置**
*   **区分逻辑**：英语是 "I study Chinese *in Beijing* *today*"。中文必须是“我今天在北京学中文”。状语后置是初学者最难改的顽疾。

**3. [G_Question_VnotV] 正反问句结构**
*   **区分逻辑**：超越简单的“吗”字句，能听懂并使用“去不去”、“好不好”。

**4. [V_MeasureWords_Basic] 基础量词必现**
*   **区分逻辑**：英语 "two books"，中文必须是“两本书”。遗漏量词或全部用“个”代替（如“两个书”），说明停留在 HSK 1。

**5.[G_Modal_Can] 能愿动词辨析（会 vs. 能 vs. 可以）**
*   **区分逻辑**：英语全用 "can"。中文里“我会游泳”(技能)、“我能游1000米”(能力)、“这里可以游泳”(许可)。这是极佳的考点。

**6. [P_Tone_Sandhi] 基础连读变调（语音KC）**
*   **区分逻辑**：ASR（语音识别）检测用户读“你好(nǐ hǎo)”时，是否自然变成了“ní hǎo”（三声变调），以及“不”和“一”的变调。

### 第二梯队：区分“初级”与“中级”（HSK 3 级，最重要的分水岭）
*这是整个评测中最关键的区域。绝大多数学习者卡在这个瓶颈期。*

**7.[G_Particle_Le_Action] 动态助词“了”（动作完成）**
*   **区分逻辑**：“了”不等于过去时！“昨天我买一本书”(❌) vs “昨天我买**了**一本书”(✅)。测试用户是否具备“体（Aspect）”的概念。

**8. [G_Particle_Guo] 经验助词“过”**
*   **区分逻辑**：区分“去了中国（went）”和“去过中国（have been to）”。

**9. [G_Structure_Ba] “把”字句（中级试金石）**
*   **区分逻辑**：中文独有结构。当 Agent 问“你的作业呢？”，用户如果回答“我把作业写完了”，直接判定达到 HSK 3 级以上；如果只说“我写完作业”，则较平庸。

**10. [G_Complement_Result] 结果补语**
*   **区分逻辑**：英语是单个动词(understand, see)，中文是“动+果”(听懂、看见、做好)。能否用出“听懂了没有”，极具区分度。

**11. [G_Complement_Direction_Simp] 简单趋向补语**
*   **区分逻辑**：动词 + 来/去（如：进来、出去、拿来）。

**12. [G_Comparison_Bi] “比”字句**
*   **区分逻辑**：英语是 "A is bigger than B"。中文是 "A 比 B 大"（S + 比 + N + Adj）。老外极易说成“A 是大比 B”(❌)。

**13. [G_Duration_Action] 动量/时量表达**
*   **区分逻辑**：表达“我学中文学了两年”。老外容易说成“我学中文两年”(❌)，动词重叠或宾语位置是极好的测试点。

### 第三梯队：区分“中级”与“中高级”（HSK 4 级）
*能掌握这些 KC 的用户，已经具备了在中文环境中生活和初步工作的能力。*

**14.[G_Structure_Bei] “被”字句（被动语态）**
*   **区分逻辑**：中文被动句带有“不如意/受损”的感情色彩（如“钱包被偷了”）。看用户是否能在描述倒霉经历时自然触发。

**15. [G_Complement_Potential] 可能补语**
*   **区分逻辑**：V + 得/不 + 结果/趋向。比如“听得懂 / 吃不下”。这是中高级学习者的标志性产出。

**16.[G_Complement_Direction_Comp] 复合趋向补语**
*   **区分逻辑**：V + 上/下/进/出 + 来/去（如：“拿出来”、“走进去”）。

**17.[G_Structure_ShiDe] “是……的”强调句**
*   **区分逻辑**：描述已发生的动作的细节（时间、地点、方式）。例如：“你是怎么来北京的？” 必须回答“我是坐飞机来的”，而不是“我坐飞机来”。

**18. [G_Conjunction_Paired] 成对关联词的熟练度**
*   **区分逻辑**：不但...而且...；连...都/也...。特别测试“连...都”（如：他连苹果都不吃），这需要高度的语感。

**19. [PR_Softener_Youdian] 语用：负面评价的委婉语“有点儿”**
*   **区分逻辑**：英语母语者容易说“这个菜很辣”。地道的中文（以及高情商的表达）是“这个菜**有点儿**辣”。这是极好的语用（Pragmatics）测试点。

### 第四梯队：区分“中高级”与“母语级/高级”（HSK 5-6 级）
*测试用户是否不仅能把话“说对”，还能“说得地道、有文采”。*

**20. [G_Topic_Comment] 话题说明结构（宾语前置）**
*   **区分逻辑**：打破 SVO。例如：“那部电影，我已经看过了。”将宾语提至句首作为话题。这是母语者极常用的句型。

**21. [G_Rhetorical_Question] 反问句式**
*   **区分逻辑**：“难道……吗？”、“怎么能……呢？”。在 Agent 与用户进行辩论或表达惊讶时触发。

**22. [V_Formal_Written] 书面语与口语的切换（语体意识）**
*   **区分逻辑**：在模拟“商务面试”场景中，是否能将“帮忙”换成“协助”，将“现在”换成“目前”，将“所以”换成“因此”。

**23.[V_Idiom_Contextual] 语境成语调动**
*   **区分逻辑**：不强求生僻成语，但能否在对话中自然使用“莫名其妙”、“乱七八糟”、“半途而废”等高频四字词语。

**24.[G_Adverb_Tone] 语气副词的精准投放**
*   **区分逻辑**：如“竟然、偏偏、居然、索性”。（如：“没想到他**竟然**没来！”）这代表了顶级的语感。

**25. [PR_Discourse_Fillers] 语用：地道的话语标记（Fillers）**
*   **区分逻辑**：在思考时，是用英语的 "um...", "like..."，还是能自然使用中文的垫音“那个……”、“就是……”、“怎么说呢”、“其实……”。这直接决定了系统给出的“Fluency（流利度）”评分。

### 💡 如何在你的 Agent 系统中应用这 25 个 KC？（配置建议）

1. **破冰题库（显性测试）**：让 Agent 随机抛出包含 `[G_Adjective_Predicate]`（形谓句） 和 `[G_Particle_Le_Action]`（“了”）的引导话题。如果这两关全挂，直接锁定为 HSK 1-2 级，结束战斗，不需要浪费 token 测高级语法。
2. **陷阱提问（隐性自适应）**：为了测试 `[G_Structure_Ba]` (“把”字句)，Agent 可以刻意提问：*“我的桌子太乱了，你能告诉我应该怎么整理这些书和笔吗？”* 迫使对方输出“你应该**把**书放在...”。
3. **评分权重**：在这 25 个 KC 中，语法 KC（G_开头）的判定置信度最高。一旦用户能准确输出“把”字句、“被”字句或可能补语，系统应大幅度拉升其“预计水平（Estimated Level）”分数。
